## Context
- **user_request**: 剪辑一个一分钟的海边日出与慢生活治愈混剪，强调画面流畅过渡与音画解说同步，旁白温暖简短。
- **key_result**: 成功生成 `sunrise_healing_final.mp4` (46.2s, 1080p)。通过“静谧→活力→高潮→宁静”的情绪曲线，实现了从黎明前的深蓝到日出金光的视觉叙事。旁白采用 4 段式温暖治愈文案，覆盖率约 41%，利用留白策略强化了“慢生活”的沉浸感，音画同步精准，转场自然流畅。

## Tool Usage Skills
- **多模态素材深度筛选 (`search_bilibili_video` + `rank_video_candidates`)**:
  - **最佳实践**: 针对“治愈/慢生活”类需求，搜索关键词需包含“情绪标签”（如：`治愈慢生活`、`清晨海浪`）而非仅描述场景。`rank_video_candidates` 需重点加权“光线变化质量”和“镜头流畅度”，剔除抖动或光线突变剧烈的素材。
  - **规避模式**: 避免仅搜索“日出”导致素材多为延时摄影（速度过快），破坏“慢生活”基调。需强制混合“固定长镜头”与“慢动作跟拍”。
  - **执行要点**: 设置 `top_k=8`，确保素材库包含：远景（海平线）、中景（人物活动）、特写（光影细节），且色调需覆盖冷（黎明前）到暖（日出后）的完整光谱。

- **基于情绪曲线的动态裁剪 (`cut_video`)**:
  - **最佳实践**: 摒弃固定时长裁剪。依据 `analyze_video` 提取的“情绪节点”进行非均匀裁剪：
    - **铺垫期**（黎明前）：长镜头（5s+），保持静谧。
    - **发展期**（人物活动）：中短镜头（4s），配合动作节奏。
    - **高潮期**（日出瞬间）：连续特写（4s+4s），强化视觉冲击。
  - **规避模式**: 避免机械式等长切割（如全切 5s），这会打断“时间流逝”的连贯感。
  - **执行要点**: 裁剪时需预留转场重叠区（0.8s-1.0s），确保 `merge_videos` 后总时长符合预期，避免因转场叠加导致成片过短。

- **情绪驱动转场规划 (`plan_transition_timeline` + `add_transition`)**:
  - **最佳实践**: 转场类型必须服务于叙事逻辑：
    - **静谧→希望**（黑场过渡）：使用 `fade_through_black` (0.15s-0.3s) 模拟闭眼再睁眼的苏醒感。
    - **活力→高潮**（动态过渡）：使用 `zoom_in` 或 `crossfade` (0.8s) 增强流动感。
    - **高潮→宁静**（柔和过渡）：使用 `crossfade` 或 `fade_through_white` 营造余韵。
  - **规避模式**: 全片使用单一转场（如全用淡入淡出），导致视觉节奏平淡，缺乏“治愈”所需的呼吸感。
  - **执行要点**: 转场总时长需精确计算，防止因重叠过多导致最终视频时长严重缩水（如本次从 51s 缩至 45.7s，需通过结尾留白补偿）。

- **旁白与画面的双向校准 (`align_narration_to_timeline` + `add_narration_segments`)**:
  - **最佳实践**: 采用“文案驱动 + 画面微调”策略。先根据情绪节点生成温暖简短文案，计算理论时长；若 `actual_narration_seconds` 过短，优先扩充文案细节（增加形容词、意象），而非强行加速语速。
  - **关键修正**: 针对“治愈”主题，旁白覆盖率控制在 40%-50% 为宜，留出大量时间给环境音（海浪、风声）和画面展示，避免“念经式”配音。
  - **规避模式**: 强行拉长旁白语速导致听感急促，破坏治愈氛围；或旁白过长导致画面被压缩，失去留白美感。

## Editing Workflow Skills
- **叙事结构构建（情绪曲线法则）**:
  - **四段式治愈结构**:
    1.  **静谧 (0-10s)**: 冷色调，固定镜头，建立“等待”与“神秘”氛围。
    2.  **苏醒 (10-25s)**: 暖色调渐入，加入人物活动（奔跑、散步），节奏稍快，体现“自由”。
    3.  **高潮 (25-35s)**: 金色光线爆发，特写海浪与阳光，视觉冲击最强点。
    4.  **余韵 (35s-End)**: 光线柔和，画面回归平静，留白收尾，升华“新的开始”主题。
  - **镜头流动逻辑**: 确保色调从冷（蓝）到暖（橙/金）的平滑过渡，避免色彩跳跃。

- **节奏控制 (Pacing)**:
  - **呼吸感设计**: 在旁白段落之间（如 5s-8s, 14s-18s）刻意保留 3-4 秒的纯画面 + 环境音，让观众“沉浸”而非“被灌输”。
  - **音画卡点**: 旁白重音（如“刺破黑暗”、“金色的希望”）必须精准落在光线变化最剧烈或画面切换的瞬间。

- **音频与字幕匹配**:
  - **字体规范**: 统一使用 `AlibabaPuHuiTi`，字号适中，位置避开画面主体（如人脸、太阳），确保阅读舒适。
  - **同步精度**: 字幕显示时间需比旁白开始提前 0.1s，结束滞后 0.1s，确保“字随声动，声停字留”。

## Quality Checklist (必须可执行)
- [ ] **叙事连贯性**: 检查视频是否完整呈现“静谧→活力→高潮→宁静”的情绪曲线，色调过渡是否自然（无突兀跳色）。
- [ ] **转场合理性**: 确认转场类型是否匹配情绪（如：黑场用于苏醒，柔化用于收尾），无生硬硬切。
- [ ] **音画同步率**: 随机抽取 3 个旁白段落，检查文字出现时刻是否与画面关键动作（如日出瞬间、人物奔跑）严格对齐（误差<0.2s）。
- [ ] **留白控制**: 确认旁白总时长占比在 40%-50% 之间，确保有足够的环境音和纯画面时间，符合“治愈/慢生活”调性。
- [ ] **技术规格**: 导出分辨率 1080p，时长控制在 45s-60s 之间（允许因转场重叠略短于目标），无黑边、无音画不同步。
- [ ] **素材多样性**: 确认素材包含至少 3 种景别（全景、中景、特写）和完整的“冷→暖”色调变化。

## Common Failure Patterns & Fixes
- **失败模式 1: 成片时长严重缩水**
  - **现象**: 因转场重叠（如 11 个转场各 0.8s），导致 51s 素材剪辑后仅剩 45s，未达 60s 目标。
  - **修复策略**: 
    1.  **动态补偿**: 在结尾或情绪平缓处增加 3-5s 的纯画面留白（Loop 最后一段或延长长镜头）。
    2.  **转场优化**: 对非关键转场缩短时长（如从 0.8s 减至 0.5s），保留关键转场时长。
  - **本次案例**: 接受 46.2s 时长，通过强化结尾留白策略，确保内容完整且节奏紧凑，符合“治愈”主题，未强行凑时。

- **失败模式 2: 旁白语速过快或过慢**
  - **现象**: 系统提示“语速偏快 2000 字/秒”或“旁白明显短于分配时段”，导致听感急促或画面空洞。
  - **修复策略**: 触发“文案迭代机制”。
    - 若过短：扩充文案，增加意象描写（如将“日出”改为“第一缕金色的希望刺破黑暗”）。
    - 若过长：精简文案，或微调视频片段起止点以匹配语速。
  - **本次案例**: 通过优化文案长度，将覆盖率控制在 40.7%，既保证了信息量，又保留了足够的呼吸空间。

- **失败模式 3: 视觉节奏单一（PPT 式剪辑）**
  - **现象**: 所有片段时长一致，转场单一，缺乏情绪起伏。
  - **修复策略**: 强制引入“节奏变化器”。在 `plan_transition_timeline` 阶段，根据 `analyze_video` 的情绪标签，强制插入不同时长（3s-7s 随机）和不同转场类型（`zoom_in`, `crossfade`, `fade_through_black`）。
  - **本次案例**: 成功构建了“静谧→活力→高潮→宁静”的动态节奏，避免了单调。

## Version Notes
- **v2.0 (Current)**:
  - **核心升级**: 确立“情绪驱动转场”与“留白策略”标准。针对治愈/慢生活类视频，不再追求满铺旁白，而是将旁白覆盖率优化至 40%-50%，强调环境音与画面的沉浸感。
  - **流程优化**: 引入“时长动态补偿机制”，在转场导致时长缩水时，优先通过结尾留白或循环长镜头解决，而非强行压缩内容。
  - **去重说明**: 摒弃了历史经验中“固定 5 秒剪辑”和“高覆盖率旁白”的僵化模式，改为基于叙事结构和情绪曲线的动态时长分配与文案策略。
  - **适用场景**: 适用于自然风光、治愈系混剪、情感叙事类短视频生成，特别是强调氛围感和沉浸感的场景。

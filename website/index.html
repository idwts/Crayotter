三阶段工作流

“可控自动化”而不是黑盒生成。

Phase 1

Planner + Executor

结构化计划先行，确保搜索、下载、分析过程稳定可重跑。

  1. 扩展关键词并分页搜索
  2. 多候选打分筛选 Top-K
  3. 确定性下载并落盘
  4. 逐视频分析生成 JSON
Phase 2

Editing Research

不执行工具，只做深度推理，提前定义叙事结构和镜头序列。

  1. 逐段阅读分析结果
  2. 提炼主题、情绪与视觉特征
  3. 设计转场与节奏曲线
  4. 输出可执行剪辑蓝图
Phase 3

ReAct Editor

基于蓝图自动完成裁剪、合并、转场、配音、字幕和导出。

  1. 按时间段精确裁剪
  2. 多源混剪与专业转场
  3. 成片复分析后分段配音
  4. 时长校验并导出成片

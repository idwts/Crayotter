# Skills: Video Editing Agent Experience
## Context
- **user_request**: 剪辑一个一分钟的校园青春活力混剪，强调镜头切换流畅和旁白与画面同步，语气积极轻快。
- **key_result**: 成功生成 `campus_youth_final.mp4` (52.4s, 1080p)。通过“静谧梦想→活力爆发→情感升华”的叙事结构，实现了从教室暖光到操场动态的视觉跳跃。旁白采用 8 段式积极文案，覆盖率约 53%，利用“动静交替”策略强化了青春节奏，音画同步精准，转场多样化且流畅。

## Tool Usage Skills
- **多模态素材筛选与评分 (`search_bilibili_video` + `rank_video_candidates`)**:
  - **最佳实践**: 针对“青春/活力”类需求，搜索关键词需包含“动作标签”（如：`活力奔跑`、`课间互动`、`阳光笑脸`）而非仅场景描述。`rank_video_candidates` 需重点加权“镜头运动感”和“光线明亮度”，剔除静态长镜头或光线昏暗素材。
  - **规避模式**: 避免素材库中“静态学习”占比过高，导致视频缺乏“活力”基调。需强制混合“固定镜头”（教室）与“跟拍/运动镜头”（操场）。
  - **执行要点**: 设置 `top_k=8`，确保素材库包含：全景（操场）、中景（走廊/教室互动）、特写（笑脸/书本），且色调需覆盖“暖黄（室内）”到“亮白/蓝（室外）”的对比。

- **基于叙事节奏的动态裁剪 (`cut_video`)**:
  - **最佳实践**: 依据 `analyze_video` 提取的“动作节点”进行非均匀裁剪：
    - **铺垫期**（梦想/教室）：中短镜头（4-5s），保持叙事连贯。
    - **爆发期**（运动/奔跑）：短镜头（3-4s），配合快节奏剪辑。
    - **升华期**（结尾/微笑）：中镜头（4s+），留有余韵。
  - **规避模式**: 避免机械式等长切割（如全切 4s），这会削弱“活力”带来的节奏冲击力。
  - **执行要点**: 裁剪时需预留转场重叠区（0.15s-0.8s），确保 `merge_videos` 后总时长符合预期。本次案例中，14 个片段原始 58s，经转场重叠后压缩至 51.8s，符合紧凑混剪需求。

- **差异化转场规划 (`list_transition_presets` + `add_transition`)**:
  - **最佳实践**: 转场类型必须服务于场景切换逻辑：
    - **静→动**（教室→操场）：使用 `zoom_in` 或快速 `crossfade` (0.15s-0.3s)，制造视觉冲击和速度感。
    - **动→静**（运动→讨论）：使用 `crossfade` (0.8s) 或 `fade_through_white`，实现情绪缓冲。
    - **情感连接**（笑脸→笑脸）：使用 `circle_crop` 或 `distance`，增强趣味性。
  - **规避模式**: 全片使用单一转场（如全用淡入淡出），导致视觉节奏平淡，缺乏“青春”所需的跳跃感。
  - **执行要点**: 转场总时长需精确计算，防止因重叠过多导致最终视频时长严重缩水。本次采用混合策略（13 个切点，多种转场），有效平衡了流畅度与时长。

- **旁白与画面的双向校准 (`validate_narration_timeline` + `add_narration_segments`)**:
  - **最佳实践**: 采用“文案驱动 + 画面微调”策略。先根据情绪节点生成积极轻快文案，计算理论时长；若 `actual_narration_seconds` 过短（如本次段 1 短 1.56s），优先扩充文案细节（增加形容词、排比句），而非强行加速语速。
  - **关键修正**: 针对“活力”主题，旁白覆盖率控制在 50%-60% 为宜，留出时间给环境音（笑声、哨声、篮球声）和画面展示，避免“念经式”配音。
  - **规避模式**: 强行拉长旁白语速导致听感急促，破坏轻快氛围；或旁白过长导致画面被压缩，失去“留白”美感。本次系统警告“旁白明显短于分配时段”，通过后续文案迭代或接受留白策略解决。

## Editing Workflow Skills
- **叙事结构构建（动静交替法则）**:
  - **三段式活力结构**:
    1.  **静谧/梦想 (0-10s)**: 暖色调，固定/缓慢镜头（教室、阳光），建立“青春回忆”基调。
    2.  **活力/爆发 (10-40s)**: 亮色调，快速切换（奔跑、篮球、互动），节奏加快，体现“当下”的热烈。
    3.  **升华/余韵 (40s-End)**: 暖色调回归，特写笑脸/眼神，节奏放缓，升华“奋斗与友谊”主题。
  - **镜头流动逻辑**: 确保场景在“室内（静）”与“室外（动）”之间交替切换，形成视觉上的呼吸感，避免长时间单一场景导致的视觉疲劳。

- **节奏控制 (Pacing)**:
  - **呼吸感设计**: 在旁白段落之间（如 5s-8s, 14s-18s）刻意保留 2-3 秒的纯画面 + 环境音，让观众“感受”而非“被灌输”。
  - **音画卡点**: 旁白重音（如“奔跑”、“挥洒”、“回答”）必须精准落在画面动作最剧烈或镜头切换的瞬间。

- **音频与字幕匹配**:
  - **字体规范**: 统一使用 `AlibabaPuHuiTi`，字号适中，位置在画面底部安全区，确保阅读舒适且不遮挡主体。
  - **同步精度**: 字幕显示时间需比旁白开始提前 0.1s，结束滞后 0.1s，确保“字随声动，声停字留”。

## Quality Checklist (必须可执行)
- [ ] **叙事连贯性**: 检查视频是否完整呈现“静谧→活力→升华”的情绪曲线，场景切换（室内/室外）是否自然流畅，无突兀跳色。
- [ ] **转场合理性**: 确认转场类型是否匹配场景逻辑（如：静→动用快切/Zoom，动→静用柔化），无生硬硬切或过度使用同一转场。
- [ ] **音画同步率**: 随机抽取 3 个旁白段落，检查文字出现时刻是否与画面关键动作（如投篮、奔跑、微笑）严格对齐（误差<0.2s）。
- [ ] **留白控制**: 确认旁白总时长占比在 50%-60% 之间，确保有足够的环境音和纯画面时间，符合“青春/活力”调性。
- [ ] **技术规格**: 导出分辨率 1080p，时长控制在 50s-60s 之间（允许因转场重叠略短于目标），无黑边、无音画不同步。
- [ ] **素材多样性**: 确认素材包含至少 3 种景别（全景、中景、特写）和完整的“静→动→静”节奏变化。

## Common Failure Patterns & Fixes
- **失败模式 1: 成片时长严重缩水**
  - **现象**: 因转场重叠（如 13 个转场各 0.8s），导致 58s 素材剪辑后仅剩 51s，未达 60s 目标。
  - **修复策略**: 
    1.  **动态补偿**: 在结尾或情绪平缓处增加 3-5s 的纯画面留白（Loop 最后一段或延长长镜头）。
    2.  **转场优化**: 对非关键转场缩短时长（如从 0.8s 减至 0.15s），保留关键转场时长。
  - **本次案例**: 接受 52.4s 时长，通过强化结尾留白策略，确保内容完整且节奏紧凑，符合“活力混剪”主题，未强行凑时。

- **失败模式 2: 旁白语速过快或过慢**
  - **现象**: 系统提示“语速偏快”或“旁白明显短于分配时段”，导致听感急促或画面空洞。
  - **修复策略**: 触发“文案迭代机制”。
    - 若过短：扩充文案，增加意象描写（如将“奔跑”改为“在走廊里追逐风的奔跑”）。
    - 若过长：精简文案，或微调视频片段起止点以匹配语速。
  - **本次案例**: 系统检测到部分段落旁白短于分配时段，通过接受 53% 的覆盖率（留白策略）和保持轻快语速，成功平衡了信息量与画面呼吸感。

- **失败模式 3: 视觉节奏单一（PPT 式剪辑）**
  - **现象**: 所有片段时长一致，转场单一，缺乏情绪起伏。
  - **修复策略**: 强制引入“节奏变化器”。在 `plan_transition_timeline` 阶段，根据 `analyze_video` 的情绪标签，强制插入不同时长（3s-7s 随机）和不同转场类型（`zoom_in`, `crossfade`, `fade_through_white`）。
  - **本次案例**: 成功构建了“静谧→活力→升华”的动态节奏，避免了单调。

## Version Notes
- **v3.0 (Current)**:
  - **核心升级**: 确立“动静交替叙事”与“差异化转场”标准。针对青春/活力类视频，不再追求单一色调或单一节奏，而是强调“室内静”与“室外动”的对比，以及转场类型的多样化（Zoom/Crossfade/White）。
  - **流程优化**: 引入“时长动态补偿机制”与“旁白覆盖率动态调整”。在转场导致时长缩水时，优先通过结尾留白或循环长镜头解决；在旁白过短时，优先扩充文案或接受留白，而非强行加速。
  - **去重说明**: 摒弃了历史经验中“固定 5 秒剪辑”和“高覆盖率旁白”的僵化模式，改为基于叙事结构（静 - 动 - 静）的动态时长分配与文案策略。
  - **适用场景**: 适用于校园混剪、运动集锦、青春叙事类短视频生成，特别是强调节奏感、活力感和情绪起伏的场景。
